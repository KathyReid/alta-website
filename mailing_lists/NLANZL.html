NLANZ-L

This page provides some basic information about the Natural Language Processing in Australia and New Zealand Discussion List (NLANZ-L), hosted by the Centre for Language Technology at Macquarie University.

This page is a copy of the NLANZ-L welcome message for new subscribers: it outlines the purpose of the list, and tells you how to subscribe, unsubscribe, post messages, and a few other things. You may wish to bookmark this page for future reference.

Purpose of this list

The NLANZ mailing list is intended as a forum for discussion of natural language processing research activities in Australia and New Zealand. The list was set up in August 1995.

Important functionalities

Subscribing

Subscribing to this list is achieved by sending a null-subject mail message to

diego.molla-aliod@mq.edu.au

with the following body text:

subscribe NLANZ-L Your-Email-Address

where Your-Email-Address is replaced, unsurprisingly, by your email address.

Unsubscribing

To unsubscribe from the list, send a null-subject mail message to

diego.molla-aliod@mq.edu.au

with the following body text:

unsubscribe NLANZ-L

Make sure you mail this from the account under which you are subscribed.

Posting messages to the list

NLANZ-L is unmoderated: any message mailed to

diego.molla-aliod@mq.edu.au

will automatically be distributed to all participants.

Replying to messages on the list

When a message arrives from NLANZ-L, you have two choices:

  • you can reply directly to the sender (that's what happens if you hit "reply" or "r" in most mail-reading programs);
  • you can mail a reply to diego.molla-aliod@mq.edu.au to have it distributed to all participants. This option is usually what you want.

Other ListServ functionality

For information on the ListServ processor used by this list and how to use it, send the following request in the body of a mail message to diego.molla-aliod@mq.edu.au:

HELP

All requests should be addressed to diego.molla-aliod@mq.edu.au.

Questions, problems ...

If you have suggestions as to how to improve the utility of this message, or encounter any problems with the list, or you have any general suggestions or comments, please mail us.

Thank you for signing up.
