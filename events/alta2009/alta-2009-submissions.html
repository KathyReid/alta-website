ALTA 2009 Submissions

ALTA 2009 is using the EasyChair system for paper submissions.

Submitting a paper is a two-step process:

  1. If you don't already have a user account at EasyChair, sign up for one here; and then

  2. Go to the EasyChair Login Page for ALTA 2009, login, go to Submissions, select Add a submission, and follow the instructions there.
